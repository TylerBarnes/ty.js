```bash

```
## Usage
```javascript

const typography = new Typography(doelgerTheme)
```
## More on theme

Doelger uses "<a href='https://fonts.google.com/specimen/Arvo'>Arvo</a>" for headers and "<a href='https://fonts.google.com/specimen/Cabin'>Cabin</a>" for body text
