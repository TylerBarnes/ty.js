## Usage
```javascript

const typography = new Typography(irvingTheme)
```
## More on theme

Irving uses "<a href='https://fonts.google.com/specimen/Exo'>Exo</a>" for headers and "<a href='https://fonts.google.com/specimen/Yrsa'>Yrsa</a>" for body text
